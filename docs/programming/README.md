# Programming & Simulation — SINJAB

This folder documents the full software architecture of the SINJAB robot.

## Robot Model
- Eurobot Robot Model Documentation.docx
Describes URDF/SDF structure, Gazebo plugins, sensors, dynamics, TF tree, and navigation interfaces.

## Manipulation
- Eurobot Manipulation Package Documentation.docx
Describes the gripper controller, state machine, contact sensing, and ROS2 interfaces.

Together, these documents define the complete simulation and control software used in Milestones M2–M5.
